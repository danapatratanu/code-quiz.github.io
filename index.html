CODE QUIZ NIGHT

Vi är ett gäng entusiaster som älskar allt som har med webb, utveckling och kod att göra. Vi vill på ett innovativt vis skapa en avslappnad atmosfär för likasinnade. Denna kväll kan du över en öl umgås med människor som har samma intressen, samt tävla i ett prestigefyllt quiz om vem som knäcker kod bäst.
